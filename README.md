# iOS_Paintboard

Author: Shen Dong

# Intro

1. Save drawings to camera roll
2. Adjust brush sizes
3. Adjust brush colors
4. Preview selection
5. undo, redo and reset
6. Send email or message with drawing picture
7. Load saved pictures from album, or from camera
